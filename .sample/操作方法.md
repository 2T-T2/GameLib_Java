# ボタンとキーの関係
<pre>
←ボタン  Sキー | Aボタン  Jキー <br>
→ボタン  Fキー | Bボタン  Kキー <br>
↑ボタン  Eキー | Xボタン  Iキー <br>
↓ボタン  Dキー | Yボタン  Lキー <br>
Lボタン  Uキー | STARTボタン  Wキー <br>
Rボタン  Oキー | SELECTボタン Tキー <br>
</pre>

# タイトル画面
↑,↓ボタンで選択、Aボタンで決定
## Game Start
ゲーム開始
## Continue
中断セーブ(後述)からゲームを再開

# ゲーム画面
START ボタン押下でゲームを開始。 <br>
←,→ボタンで移動。 <br>
ゲームプレイ中に、STARTボタンを押すと中断セーブが行われ、タイトル画面で「Continue」を選ぶと、現在の状態からゲームが再開される。 <br>

# ゲームオーバー画面
いずれかのキーを押すとタイトル画面に戻る
